# Simple, but unique calculator in React.js
## description

## Local Setup

```sh
$ git clone https://github.com/AlmubdyMutaikhan/Frontend-React-JS-calculator.git
```

```sh
$ cd react-calculator
```

```sh
$ npm install
```

## Run Locally

```sh
$ npm start
```

## Run Tests

```sh
$ npm test
```
