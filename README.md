## description
> Sample implementation of electronic calculator on the web using React.js framework.

 
## Local Setup

```sh
$ git clone https://github.com/AlmubdyMutaikhan/Frontend-React-JS-calculator.git
```

```sh
$ cd react-calculator
```

```sh
$ npm install
```

## Run Locally

```sh
$ npm start
```

## Run Tests

```sh
$ npm test
```
